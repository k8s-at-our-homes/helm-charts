# Opentherm Monitor

## How to use

Add repository by running:

```bash
helm repo add k8s-at-our-home https://k8s-at-our-homes.github.io/helm-charts/
helm install opentherm-monitor k8s-at-our-home/opentherm-monitor
```

Or get the chart from ghcr.io:

```bash
helm install opentherm-monitor oci://ghcr.io/k8s-at-our-homes/helm-charts/opentherm-monitor
```
